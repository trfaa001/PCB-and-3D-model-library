# Shaft attachment for sim racing wheels

<!-- Project description -->
Attachment for the 130ST-M10015 shaft for attaching a 70mm quick release to the motor. The inner part is a modified design of a part I found on the internet. The outer part has been added to
hinder cables getting stuck in the screws.
